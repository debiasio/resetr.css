# resetr.css

## Installation

### Add dependency

```bash
yarn add resetr.css
```

### For Vue.js SPA project

Add `import 'resetr.css'` at the top of src/main.js

### For Nuxt.js project

Add `resetr.css` in the Global CSS array in nuxt.config.js

```javascript
// ...
css: [
  'resetr.css'
],
// ...
```
